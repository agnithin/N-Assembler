NASM

Instructions For Use

 > Create a folder named asm in C drive, then copy all the files to the dir.
 
 > The whole program should be written in caps.

 > Instn should follow a space followed by comma(,) again followed by space.

 > Spaces at the end of the pgm should be avoided.

 > Label should be written in the starting of the line followed by any number of spaces then colen(:) followed by space and instn.

 > Declarative statements should come after STOP instn.

 > Currently the program does not take advanced assembler directives as input.

 > This program optimized for 800 x 600 resolution.
